## step 1

考え方は、今のノードと次のノードを見比べて、重複していたら削除するというものです。  
削除の書き方が分からず、delete や append などを試しに書いてみました。  
```Python
# 最初に自分で書いた誤ったコード
class Solution:
    def deleteDuplicates(self, head: Optional[ListNode]) -> Optional[ListNode]:
        visited = set()
        now_node = head
        next_node = head
        while now_node is not None:
            if now_node in visited:
                return now_node
            visited.add(now_node)
            now_node = now_node.next
            next_node = now_node.next.next
            if now_node == next_node in visited:
                visited.append(now_node)
                return next_node
```
